**html基础标签简单记录，以防自己忘记**

1.html基本结构 <br>
```html
<html>
    <head>
        <title>网页标签的名字</title>
    <head>
    <body> 
        网页显示内容
    </body>
<html>
```
<br>

2.head中标签
```html
设置编码    <meta charset="utf-8">
网页标签的名字  <title> </title>
网页的描述      <meta name="description" content="java哈哈哈哈"/>
网页关键字      <meta name="keywords" content="java, IT"/>
名字        <meta name="author" content="网易"/>
定时刷新进入其他网页,从当前网页三秒进入百度   <meta http-equiv="refresh" content="3;https://www.baodu.com"/>
清除网页缓存    <meta http-equiv="Cache-Control" content="no-Cache" />
```

3.基本标签
```html
段落标记    <p> </p>
标题    <h1> </h1> ~  <h6> </h6>
换行    <br>
斜体    <em> </em>
插入图片    <img src="路径" alt="图片描述">
超链接     <a href="路径">name</a>
```
